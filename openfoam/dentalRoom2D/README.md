# `dentalRoom2D` — foam-extend 4.1 reactingParcelFoam reference case

This is **`Sweep_Case_03`** from the ELGIN paper's $(V_{\rm in},\,U_{\rm
mag})$ factorial design space, shipped as the single demonstration case
that accompanies the public ELGIN checkpoint.


## Quick facts

| Quantity | Value |
|---|---|
| Solver | `reactingParcelFoam` (foam-extend 4.1) |
| Domain | $4\,\mathrm{m}\times 3\,\mathrm{m}\times 0.01\,\mathrm{m}$ pseudo-2D dental room |
| Obstacles | Dentist ($x\in[1.40,1.60]$, $y\in[0,1.40]$) and Patient ($x\in[2.40,2.60]$, $y\in[0,0.80]$) |
| Mesh | 8 000 raw cells, **7 704 active fluid cells** after obstacle carve-out |
| Ceiling supply inlet | $x\in[1.90,2.10]\,\mathrm{m}$, $V_{\rm in}=0.10\,\mathrm{m\,s^{-1}}$ downward |
| Side outlet | right wall, $y\in[1.40,1.60]\,\mathrm{m}$ (pressure outlet) |
| Aerosol injection | $(2.40,\,0.90,\,0.005)\,\mathrm{m}$ — patient oral cavity, horizontal toward dentist |
| Jet exit speed | $U_{\rm mag}=30\,\mathrm{m\,s^{-1}}$, half-angle $\theta=20^{\circ}$ |
| Droplet PDF | Rosin–Rammler, $d_p\in[1,50]\,\mu\mathrm{m}$, $\overline{d_p}=20\,\mu\mathrm{m}$, $n=2$ |
| Duration | 30 s, adaptive $\Delta t\leq 0.01$ s, $\mathrm{maxCo}=0.3$ |
| Wall-clock | $\approx 40$ min on a single CPU core |

## Two-phase Allrun pipeline

The `Allrun` script runs a sequenced two-phase pipeline (paper Sec. II):

1. **Phase 1 — `simpleFoam`** — steady-state incompressible $k$–$\omega$
   SST RANS to converge a frozen carrier flow.
2. **Transition** — `scripts/init_reacting_fields.py` rewrites the
   converged time directory with compressible-side BCs and seeds
   $T$, $\mu_t$, $\alpha_t$, $Y_{\mathrm{H_2O}}$, $Y_{\mathrm{N_2}}$ and
   $G$ for Phase 2.
3. **Phase 2 — `reactingParcelFoam`** — transient compressible
   Navier–Stokes coupled to the reacting parcel cloud, with Wells'
   $d^{2}$-law evaporation via the `LiquidEvaporation` sub-model.
4. **Post-process** — `foamToVTK` followed by
   `scripts/compute_bze.py` for the breathing-zone exposure time series.

The `system/{controlDict,fvSchemes,fvSolution}.{rans,lagrangian}`
splits are swapped into the canonical filenames by `Allrun` between
phases.

## What's in this folder

| Path | Purpose |
|---|---|
| `0.orig/` | bare initial conditions for $U$, $p$, $k$, $\omega$, $\nu_t$ (Allrun copies them to `0/`) |
| `constant/polyMesh/blockMeshDict` | structured 25-block mesh definition with carved-out dentist + patient obstacles |
| `constant/polyMesh/{faces,owner,neighbour,points,boundary}` | pre-built mesh files (regenerated by `blockMesh` if `Allrun` is re-run) |
| `constant/reactingCloud1Properties` | Lagrangian cloud: ConeInjection + SphereDrag + StochasticDispersionRAS + LiquidEvaporation (Wells $d^2$-law) |
| `constant/thermophysicalProperties` | air thermophysics with H$_2$O vapour mixture |
| `constant/transportProperties` | kinematic viscosity for the Phase 1 incompressible run |
| `constant/turbulenceProperties` / `constant/RASProperties` | $k$–$\omega$ SST RANS selector |
| `constant/chemistryProperties`, `constant/reactions` | reactions table (no active reactions; required by reactingParcelFoam) |
| `constant/radiationProperties` | radiation off (required by reactingParcelFoam) |
| `constant/g` | gravity vector $(0,-9.81,0)\,\mathrm{m\,s^{-2}}$ |
| `chemkin/{foam.dat,foam.inp}` | minimal Chemkin chemistry definition (single non-reacting mixture) |
| `system/controlDict` | run-time control (`Allrun` swaps `.rans` vs `.lagrangian` variants into this slot) |
| `system/{controlDict,fvSchemes,fvSolution}.{rans,lagrangian}` | phase-specific control / discretisation / solver settings |
| `scripts/init_reacting_fields.py` | Phase 1 → Phase 2 field transition utility |
| `scripts/compute_bze.py` | breathing-zone exposure post-processor |
| `Allrun` | end-to-end pipeline: `blockMesh` → `checkMesh` → `simpleFoam` → transition → `reactingParcelFoam` → `foamToVTK` → BZE |
| `Allclean` | remove all generated files |

## Running the case

```bash
source $FOAM_INST_DIR/foam-extend-4.1/etc/bashrc

cd openfoam/dentalRoom2D
./Allrun          # full two-phase pipeline + post-processing
```
